# ATM Machine Simulation - COAL Project

## Project Description
This is an ATM machine simulation implemented in x86 Assembly language using MASM and Irvine32 library.

## Features
- PIN authentication system
- Balance checking
- Deposit functionality
- Withdrawal with validation
- Menu-driven interface

## Requirements
- MASM Assembler
- Irvine32 Library
- Visual Studio

## How to Run
1. Assemble: `ml /c /coff ATMMachine.asm`
2. Link: `link /SUBSYSTEM:CONSOLE ATMMachine.obj irvine32.lib`
3. Run: `ATMMachine.exe`
